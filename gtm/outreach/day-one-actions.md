## Action 2 — SE UK: 10 names from Companies House (45 min)

**Why:** Private owner-operators in SE UK are specifically not findable via web search. They're in public company records.

1. Go to find-and-update.company-information.service.gov.uk
2. Search: "industrial properties" → select "Companies" → filter active
3. Sort by registered address → look for postcodes TN, ME, CT (Kent), GU, KT (Surrey), CM, SS (Essex)
4. Open each result → check SIC code = 68100 or 68209
5. Click on a director name → Companies House shows ALL their companies
6. Anyone running 3+ property companies = prospect → note name
7. Google "[Director Name] + LinkedIn" → find their profile
8. Add to `prospects-seuk.csv`

**Target: 10 names in 45 minutes. Start with Kent (TN/ME postcodes) — highest industrial density.**
