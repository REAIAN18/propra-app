## How to pull names (20 min method)

**Companies House — fastest route:**
1. Go to: `find-and-update.company-information.service.gov.uk`
2. Search: `"industrial properties"` or `"commercial estates"` → filter by postcode below
3. Open result → check SIC code (want 68100 = Buying/Selling Own Real Estate, 68209 = Other Let/Manage Own Property)
4. Click director name → CH shows all their companies → **3+ SIC 68100/68209 = portfolio owner**
5. Note: Director name, all associated property companies, registered address

**LinkedIn validation:**
- Search `"[surname] properties" + [county]` — confirms owner/director role
- Check current position + number of connections — active owners tend to have 200–500+ connections in property circles

**Email hunt:**
- Hunter.io: enter company domain (from CH)
- RocketReach: enter name + company
- Company website (many owner-ops have simple websites with "Contact" pages)

## High-Priority Signal — Director Cross-Reference

Any director showing:
- **3+ SIC 68100/68209 companies registered** → confirmed portfolio owner (assets split across LLCs — standard UK tax structure)
- **Company names like "[Surname] Properties", "[Area] Commercial", "[Name] Industrial Holdings"** → owner-managed, not fund-managed
- **Last accounts filed within 12 months** → active, not dormant
- **Fixed assets on balance sheet £5M–£50M** → in our commission sweet spot
