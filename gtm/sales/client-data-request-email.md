---

**What I need from you:**

**1. Asset list**
Address, property type (office / industrial / retail / other), approximate floor area for each asset.
*Why: This is the base layer. Everything else — benchmarks, income modelling, planning checks — runs off it.*

**2. Insurance**
Current broker name, annual premium, and renewal date (approximate is fine).
*Why: This is almost always the fastest saving. I need to know your renewal window to time the retender correctly.*

**3. Energy**
Current supplier per property (or overall), annual energy spend, and contract end date if you have it.
*Why: Tariff benchmarking runs in real time. Knowing your contract end date tells us whether we can switch now or position you for the next window.*

**4. Rent roll**
For each tenanted unit: tenant name, lease expiry, current passing rent, and ERV if you have it.
*Why: This is where most portfolios leak the most. I want to surface every upcoming review window and every gap to ERV before we talk next.*

**Optional (speeds things up):**
A rent schedule PDF or recent accounts if you have them to hand. Not essential — but it saves a round trip.

---

**How to send it:**

Email is fine. A shared folder (Google Drive, Dropbox, whatever you use) works too. Whatever is easiest.

The model will show you exactly where the gaps are, what each fix is worth, and what the order of attack should be.
