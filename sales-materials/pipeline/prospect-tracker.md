## How to Use This Tracker

**When to update:**
- After sending Email 1 → set `Email 1 Sent` date, set `Status: Sent`
- When a reply arrives → set `Reply?` to Y, add brief note, move `Status: Replied`
- When demo booked → set `Demo Booked` to Y and `Demo Date`
- After the demo → move `Status: Demo Done`, note the outcome
- After sending engagement letter or next-step proposal → `Status: Proposal Sent`
- When they sign on → `Status: Won`
- When they explicitly decline / no contact after 3 follow-ups → `Status: Lost` or `Nurture`

**Status values:**
| Status | Meaning |
|---|---|
| `Sent` | Email 1 sent, awaiting reply |
| `Replied` | They replied — action required |
| `Demo Booked` | Call scheduled |
| `Demo Done` | Demo completed, next step pending |
| `Proposal Sent` | Engagement letter / proposal sent |
| `Won` | Client signed on |
| `Lost` | Declined or unresponsive after full sequence |
| `Nurture` | Not now but worth re-engaging in 60–90 days |

**Pre-send actions:** Most Wave 1 and Wave 2 prospects require a name lookup before sending. See the `Pre-send action` column. Do the lookup, update the Name and Email fields, then send.

**Sequence cadence:** Day 0 → Email 1. Day 3 → LinkedIn DM. Day 7 → Email 2 follow-up. See `fl-owner-operator-sequence.md` for full sequence copy.

## Daily Checklist (run every morning, 5 min)

**1. New replies?**
- Check inbox for any responses to Wave 1 / Wave 2 emails
- If replied: update `Reply?` to Y, update `Status` to `Replied`, add note
- Book demo immediately if they're interested — don't delay more than 2 hours

**2. Demos today or tomorrow?**
- Check `Demo Date` column for anything in the next 48 hours
- Pull up pre-call prep checklist in `sales-materials/demo/call-script.md`
- Look up their assets on county appraiser site night before

**3. Follow-ups due?**
- Count days since `Email 1 Sent` for each `Sent` status row
- Day 3: send LinkedIn DM (see `fl-owner-operator-sequence.md`)
- Day 7: send Email 2 follow-up (see `fl-owner-operator-sequence.md`)
- Day 14: final touchpoint, then move to `Nurture` if no response

**4. Pre-send actions outstanding?**
- Any rows with `[TBD]` in the Name column = names still to look up
- Target: do one batch lookup session (90–120 min) to populate 10–15 names
- Once name + email confirmed: send Email 1, set date, update Status to `Sent`

**5. Won / proposal stage?**
- Any `Proposal Sent` rows: follow up if more than 5 business days since sending
- Any `Demo Done` rows with no next step set: send engagement letter within 24hrs
